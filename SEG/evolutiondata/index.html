Helix

Helix is a collection of open source Java software systems and their associated release histories in the form of the distributed libraries (.jar files containing compiled bytecode), accompanied by the metrics and metadata that have been extracted as a result of reverse-engineering the bytecode contained within the libraries.

Helix consists of 41 different systems that are actively maintained and used, each with a rich release history.
