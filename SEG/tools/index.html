Mutations

The Mutations tool is a software evolution analysis tool that produces reports regarding the nature of growth and change throughout a software systems history.

Mutations works by extracting various metrics and metadata directly from reverse-engineered Java bytecode from the release .jar files for a software system and utilising this data to attempt to explain the evolution of the system.
