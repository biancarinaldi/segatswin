Get Source Code

The source code for Mutations resides within a Bazaar repository hosted on Launchpad.

In order to acquire the source code, you must have the Bazaar version control system installed. If you have not done so already, Download Bazaar and Install it on your machine.

Once Bazaar is installed, you can checkout a copy of Mutations using the command:

bzr checkout https://code.launchpad.net/~swincsse/mutations/trunk
