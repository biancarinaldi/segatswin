Helix is a organised collection of open source Java software systems and their associated release histories made available for the purposes of conducting empirical research in the area of software evolution.

Currently, we have 41 systems that form Helix, encompassing various different types of open source Java software systems. The distribution package for each of the systems we have available contains:

  • Raw Data (compiled Java bytecode packaged in .jar files) for each of the systems releases
  • Metric Data extracted from the system (compiled Java bytecode packaged in .jar files)
  • Descriptive metadata (general information about the system and the list of significant releases)

The Mutations software evolution tool provides a means of extracting and analysing the data contained within the Helix system distributions, producing reports that portray the growth and change of the systems and attempt to explain it's nature.

Helix is controlled and maintained by the Software Evolution Group @ Swinburne University of Technology.
