Installation Guide

Installation

To install Mutations, you must take the following steps:

  1. Download Mutations
  2. Extract the contents of the Mutations archive file to a directory.
  3. Create and set the MUTATIONS_HOME environment variable to the root directory for the Mutations folder (chosen in the previous step) to the system path.
  4. Create and set the MUTATIONS_CONFIG environment variable to the Mutations.properties configuration file. If this file does not currently exist, create an empty file with that name.

Configuration

Next, you will need to set the configuration setting for the builds directory, which is where the systems you wish to run Mutations on are stored. To do this, change (or add if it does not exist) the following setting to your Mutations.properties file: buildsDirectory=<Builds Directory>

In this case, <Builds Directory> will be replaced with the path to your builds directory, for example: buildsDirectory=/Users/ajones/Development/builds

Using Mutations

Now that you have installed and configured Mutations, it should now be ready for use. For instructions on how to use Mutations, proceed to the User Manual.
